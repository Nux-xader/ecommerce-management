
# Ecommerce Management

A simple ecommerce management based on [gin-gonic](https://github.com/gin-gonic/gin)


## Installation


```bash
  git clone https://github.com/Nux-xader/ecommerce-management
  cd ecommerce-management
  go mod tidy
```
    

    
## Usage


```bash
  go run main.go
```
